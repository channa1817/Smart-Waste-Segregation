# 🚀 Smart Waste Segregation System

## 📌 Project Overview
This project is an AI + Embedded Systems based Smart Waste Segregation System that classifies waste using:

- Arduino Sensors
- ESP32 Communication
- Machine Learning (VGG16)
- OpenCV Real-time Detection

---

## 🎯 Features
- Dry/Wet/Metal waste segregation
- IR Sensor object detection
- Metal Sensor identification
- Moisture Sensor for wet waste
- Servo Motor bin control
- ESP32 serial communication
- TensorFlow VGG16 image classification
- TrashNet dataset training

---

## 🧠 Machine Learning Model
- Dataset: TrashNet
- Classes:
  - cardboard
  - glass
  - metal
  - paper
  - plastic
  - trash
- Training Images: 2024
- Validation Images: 503
- Model: VGG16 Transfer Learning
