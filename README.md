# 🚀 Smart Waste Segregation System

## 📌 Project Overview
This project is an AI + Embedded Systems based Smart Waste Segregation System that classifies waste materials using machine learning, sensors, and automation.

---

## 🎯 Features
- Dry/Wet waste segregation
- Metal waste detection
- IR Sensor object detection
- Moisture Sensor analysis
- Servo Motor automation
- Arduino integration
- ESP32 communication
- TensorFlow VGG16 image classification
- OpenCV real-time waste detection

---

## 🧠 Machine Learning Model
### Dataset:
TrashNet Dataset

### Waste Classes:
- cardboard
- glass
- metal
- paper
- plastic
- trash

### Training Details:
- Training Images: 2024
- Validation Images: 503
- Model: VGG16 Transfer Learning
- Platform: Google Colab
